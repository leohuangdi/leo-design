VIAS project

VIAS system

VIAS is a totally new system for security and automation. It includes everything you need for your house or business: Video monitoring, Automation, Alarm system and Management software. The main difference with the other systems is scalability: You can extend the system by adding more devices, more partitions, more users, more gateways without limit.
